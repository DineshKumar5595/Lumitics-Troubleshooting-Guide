# Lumitics-Troubleshooting-Guide
Interactive Troubleshooting Guide for Lumitics IoT Devices

Key Features
   Searchable Symptoms: Quickly filter through issues using the live search bar.
   Intelligent Logic: Cross-linked diagnostic paths that bridge between different symptoms (e.g., RFID issues branching into Power issues).
   History Tracking: A "Back" button to correct mistakes without restarting the whole process.
   Visual Interface: Custom UI design based on a user-centric oval-button prototype.
   
🛠️ Current Diagnostic Paths
   RFID Card Not Detected: Troubleshooting sensor detection and card configuration.
   Not Receiving Data: Checking network connectivity and portal status.
   Device Not Turning On: Power supply and battery indicator diagnostics.
   LED Blinking Red: Hardware-level error state resolutions.
